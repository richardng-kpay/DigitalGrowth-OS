## Conventions

- **Status:** Draft → In Review → Approved → Live → Ended
- **OKR linkage:** Reference the specific KR in `TEAM-GOALS.md` this campaign is meant to move
- **Owner** = the person accountable for delivery; contributors are listed in the brief
- Run `/weekly-performance-report` to pull active campaign KPIs into the weekly report
